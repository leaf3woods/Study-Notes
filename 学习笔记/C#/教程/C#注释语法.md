## C# 注释语法

**注释语法**

​    为了使用C＃提供的XML注释功能，你的注释应该使用特殊的注释语法（///）开头。在///之后，你可以使用预先定义的标签注释你的代码，也可以插入你自己定义的标签。你定制的标签将会在随后加入到生成的注释文档中。

```c#
/// <summary>
/// Sample Hello World in C#
/// </summary>
public class HelloWorld
{
	/// <summary>
	/// Sample Hello World in C#
	/// </summary>
	public static void Fun()
	{
```
